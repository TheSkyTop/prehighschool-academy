# PreHighSchool Academy

A Year 4-6 selective and scholarship readiness app with weekly lessons, demos, quizzes, rewards, and mistake-bank review.

## Local Development

```bash
npm install
npm run dev
```

## Release Check

```bash
npm run release:check
```

## GitHub Pages

This repository includes `.github/workflows/deploy-pages.yml`. After GitHub Pages is enabled with GitHub Actions as the source, every push to `main` builds and publishes the site automatically.
